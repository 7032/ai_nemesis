# AI Nemesis

AI（大規模言語モデル）によって生成されたシューティングゲーム（STG）のコレクションプロジェクト。
往年の名作へのオマージュとして、異なるAIがコードとアセットを生成しました。

## 収録タイトル (Included Games)

### 1. Viper-Zero: The Guardian
*   **Creator:** Google Gemini 3 Pro High
*   **Description:** 古典的なルールのプレイヤーと敵の攻防を楽しめるオーソドックスなSTG。
*   **Play:** [gemini-nemesis.html](gemini-nemesis.html)

### 2. Viper-Genesis: A Tribute to Gradius
*   **Creator:** Anthropic Claude
*   **Description:** 多彩なパワーアップシステムを搭載した、某横スクロールSTGへのトリビュート。
*   **Play:** [claude-nemesis.html](claude-nemesis.html)

### 3. STARLINE VECTOR
*   **Creator:** OpenAI ChatGPT
*   **Description:** ベクターグラフィックスとプロシージャルサウンドが特徴のスタイリッシュなSTG。
*   **Play:** [chatgpt-nemesis.html](chatgpt-nemesis.html)
